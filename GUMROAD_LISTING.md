## Descripción completa (pega esto en Gumroad)

### ¿Qué es esto?

Un sitio web editorial completo sobre **finanzas personales y salud**, construido con tecnología profesional y listo para generar ingresos con Google AdSense desde el primer día.

**Demo en vivo:** https://finanzas-salud-global--angelmarte458.replit.app
**Código en GitHub:** https://github.com/raymellmarte98/vidafinanciera

---

### ✅ Qué incluye

**Sitio web completo**
- 4 páginas: Home, Finanzas, Salud, Artículo individual
- Diseño editorial premium (estilo revista de alta gama)
- 100% responsivo — móvil, tablet y escritorio
- Modo claro y oscuro automático
- Selector de idioma: **Español / English / Português**

**Backend y base de datos**
- API REST con 5 endpoints documentados
- Base de datos PostgreSQL con Drizzle ORM
- 13 artículos de ejemplo incluidos
- Sistema de newsletter con captura de emails

**Monetización AdSense**
- 4 zonas de anuncios listas (banner, rectángulo, en contenido)
- Compatible con Auto Ads y anuncios manuales
- Publisher ID configurable en un solo lugar

**Para el desarrollador**
- Código TypeScript end-to-end (sin errores de tipos)
- OpenAPI 3.1 spec incluida
- Documentación técnica completa
- Guía de instalación paso a paso

### ❓ Preguntas frecuentes

**¿Necesito saber programar?**
Para instalar el proyecto en Replit, no — basta con importar el repo. Para personalizarlo a fondo, ayuda conocer React básico.

**¿Puedo usar mi propio dominio?**
Sí. El sitio se puede desplegar en cualquier hosting: Replit, Vercel, Railway, VPS, etc.

**¿Incluye la cuenta de AdSense?**
No. AdSense es una cuenta personal — tú la creas gratis en google.com/adsense.

**¿Puedo venderlo a mis clientes?**
Sí, licencia para uso comercial incluida.
